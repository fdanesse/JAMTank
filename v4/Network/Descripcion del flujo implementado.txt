
Host crea el juego:
    Levanta el servidor configurando mapa, vidas, jugadores
        {
        'jugadores': 2,
        'vidas': 5,
        'mapa': 'f1.png'
        }

    Levanta su cliente
    Registra su cliente en el servidor configurando nick y tanque
        {
        'register': {
            'nick': 'flavio',
            'tank': 't1.png'
            }
        }

    inicia handler_registro
    inicia handler_anuncio (publica el servidor en la red)
        {
        'ip': '192.168.1.11',
        'nickh': 'flavio',
        'jugadores': 7,
        'vidas': 5,
        'mapa': 'f1.png'
        }

    * handler_registro Recibe:
        {
            "aceptado": True,
            "game": {
                "jugadores": 2,
                "vidas": 5,
                "mapa": "f0.png",
                "todos": False,
                },
            "players": {
                "192.168.1.11": {
                    "nick": "flavio",
                    "tank": "t1.png"
                    }
                }
            }
        Envía los datos a la interfaz
        Cuando recibe "todos" == False, Reinicia handler_anuncio y envia "play-enabled" == False a la interfaz
        Cuando recibe "todos" == True, Detiene handler_anuncio y envia "play-enabled" == True a la interfaz
            Cuando desde la interfaz gráfica se lanza jugar:
                Detiene handler_registro
                Lanza el juego
        NOTA:
            Al Comenzar el juego, se envia al server: _dict = {"ingame": {"nada": ""}}
            Cuando el servidor recibe esto, agrega: self._dict_game['run'] = True
            Cuando los clientes no host reciben esto en handler_registro, emiten "play-run" a la interfaz gráfica, la cual lanza el juego


Cliente no host:
    Escucha la red buscando juegos creados
    Al encontrar un juego, informa a la interfaz y el juego se agrega a una lista
    Cuando el usuario selecciona uno de esos juegos y se conecta:
        Detiene la escucha en la red
        Levanta su cliente
        Registra su cliente en el servidor configurando nick y tanque
        inicia handler_registro
        Cuando recibe _dict_game['run'] = True handler_registro y lanza el juego
