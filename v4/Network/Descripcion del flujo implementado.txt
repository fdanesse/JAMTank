
Host crea el juego:
    Levanta el servidor configurando mapa, vidas, jugadores
    Levanta su cliente
    Registra su cliente en el servidor configurando nick y tanque

    inicia handler_registro
    inicia handler_anuncio (publica el servidor en la red)

    * handler_registro Recibe:
        {
            "aceptado": True,
            "game": {
                "todos": False, "jugadores": 2, "vidas": 5,
                "mapa": "f0.png"
                },
            "players": {
                "192.168.1.11": {
                    "nick": "flavio",
                    "tank": "t1.png"
                    }
                }
            }
        Env√≠a los datos a la interfaz
        Cuando recibe "todos" == True, Cierra el servidor deteniendo handler_anuncio y habilita jugar en la interfaz
            Cuando seleccione jugar en la interfaz, debe enviar running al server para que todos lancen el juego
        Cuando recibe "running" == True lanza el juego



Cliente no host:
    Escucha la red buscando juegos creados
    Al encontrar un juego, informa a la interfaz y el juego se agrega a una lista
    Cuando el usuario selecciona uno de esos juegos y se conecta:
        Detiene la escucha en la red
        Levanta su cliente
        Registra su cliente en el servidor configurando nick y tanque
        inicia handler_registro
        Cuando recibe "running" == True lanza el juego
